============================================================
  DeepSeek 微信自动回复系统 v2.0  使用说明
============================================================

【适用环境】
  - Windows 10 / 11
  - 微信PC版 4.1.9.30 及以上
  - Python 3.9+
  - Ollama (本地部署 DeepSeek 模型)

============================================================
  一、安装步骤
============================================================

1. 安装 Python 依赖（打开终端，进入本目录）:

   pip install -r requirements.txt

   (如果下载慢，先用阿里源:
    pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/)

2. 安装 Ollama 并下载模型:

   官网: https://ollama.com/download
   安装后打开终端运行:

   ollama pull deepseek-r1:7b

   (7b 是轻量版，显卡好的可以用 deepseek-r1:14b)

3. 可选 - 安装 OCR 支持 (消息读取更准确):

   pip install paddleocr

============================================================
  二、配置
============================================================

1. 编辑 names.txt，写入要监听的好友名称（每行一个）:

   张三
   李四
   王五

2. 如果模型不是 deepseek-r1:7b，编辑 config.py 修改:

   MODEL_NAME = '你的模型名'    (用 ollama list 查看)

3. 其他可选配置 (config.py):

   POLL_INTERVAL = 1.0   消息检查间隔(秒)
   REPLY_DELAY = 0.5     回复延迟(秒)，模拟真人
   TEMPERATURE = 0.7     AI 回复创意度

============================================================
  三、运行
============================================================

1. 确保 Ollama 在运行:

   ollama serve

2. 确保微信PC版已登录，窗口处于打开状态

3. 在本目录打开终端，运行:

   python app.py

4. 停止: 按 Ctrl+C

============================================================
  四、消息读取模式说明
============================================================

本程序有三种消息读取方式，按优先级自动切换:

  [默认] 剪贴板模式 — 点击消息→Ctrl+C→读剪贴板
          ✅ 无需额外安装
          ✅ 所有微信版本通用
          ⚠️ 需要微信窗口可见

  [备选] UIA 模式 — 直接读微信界面控件
          ✅ 最快
          ⚠️ 随微信版本变化可能失效

  [可选] OCR 模式 — 截图→OCR识别
          ✅ 最鲁棒，不依赖微信UI结构
          ⚠️ 需安装 paddleocr 或 easyocr
          ⚠️ 速度较慢

  启用OCR模式: 编辑 config.py → USE_OCR_FALLBACK = True

============================================================
  五、故障排查
============================================================

Q: 提示"未找到微信窗口"？
A: 确保微信窗口已打开且未最小化到系统托盘。

Q: 读取不到消息？
A: 1. 确保聊天窗口是打开的
   2. 安装 OCR 支持: pip install paddleocr
   3. 运行诊断: python wechat_bot.py

Q: AI 回复不正常？
A: 1. 确认 Ollama 正在运行: 浏览器打开 http://localhost:11434
   2. 确认模型已下载: ollama list
   3. 检查 config.py 中 MODEL_NAME 是否正确

Q: 发送消息失败？
A: 确保微信输入法是英文/中文输入状态不影响粘贴
